Data-Theme

A data-theme will allow you to change the entire theme of the document by changing a single value.


global.css

html[data-theme="dark"], :root {
    --background-color: #0f0f0f;  
    --color: #f1f1f1;
}

The CSS selector will select html[data-theme="dark"] value from the HTML document.

The two dashes -- define a variable. These are different from attributes as they don't effect the styling, rather they define values that can be used and accessed with a common name.

Don't worry about the :root. This is used to refer to the root of the document. My CSS previewer requires me to edit the root classes of the document.

Body Margin

The body has a default margin which is roughly 8px. This can get really frustrating as a beginner and wondering why things aren't aliging properly. Don't forget to click on the checkmark in the top-right corner above the code-block to preview the current CSS.

global.css

html[data-theme="dark"], :root {
    --background-color: #0f0f0f;  
    --textColor: #f1f1f1;
}
body {
    margin: 0px;
    background-color: var(--background-color);
    color: var(--color);
}

Remember that the two dashes -- define a variable. We can assign these variable values' to CSS attributes. We can access them with var().

Fixing the Static

By default, all HTML elements will have a position of static. This means that elements will follow the flow of the document layout. Changing an element's position to fixed will make the element not repsond to any changes in the document. If you have a look at YouTube's header and left navigation section, they don't move, so we need to make them fixed.

global.css

html[data-theme="dark"], :root {
    --background-color: #0f0f0f;  
    --color: #f1f1f1;
}
body {
    margin: 0px;
    background-color: var(--background-color);
    color: var(--color);
}
#desktopHeader, #drawer {
    position: fixed;
}

Flexbox

Flexbox is - in my opinion - the holy grail of all display styles.

Display defines how an element is shown in the document, as well how their inner layouts work. Flexbox is one of these modes.

Flexbox is quite literally a flexible box. Depending on how you set it up; the size can adjust automatically and elements within the flexbox will adjust accordingly.

There are a few main topics I will focus on for flexbox:

  • justify-content
  • align-items
  • flex-basis/shrink/grow

To learn more, checkout A Guide to Flexbox by Chris Coyier

justify-content

These are the options for justify-content. These will align either the x or y axis depending on the flex-direction. By default, justify-content will effect the x axis.

justify-content: flex-start;

justify-content: center;

justify-content: flex-end;

justify-content: space-evenly;

justify-content: space-between;

justify-content: space-around;

align-items

These are some of the options for align-items. These will align either the y or x axis depending on the flex-direction. By default, align-items will effect the y axis. All the spacing functions will work the same.

align-items: flex-start;

align-items: center;

align-items: flex-end;

flex-basis/shrink/grow

flex-basis allows you to change the initial size of the flex item.

flex-shrink allows you to change the the relative shrink rate compared to other flex items.

flex-grow allows you to change the relative growth rate compared to other flex items.

flex-basis: 70px;

flex-shrink: 2;

flex-grow: 2;

Width and Height

Both width and height can have the same type of values. I'll mainly focus on:

  • max-width/height
  • min-width/height
  • viewport size
  • percentage size
  • absolute size

max-width/height and min-width/height

These are the maximum width or height of an element or can be used to set the minimum width or height of an element.

viewport sizes

These are based of the screen size of the current device.

percentage sizes

These are relative sizes based on the parent element size.

absolute sizes

These are fixed sizes that will not change.

Margins and Padding

Margins and padding help fine tune and space elements and content within elements.

Margins

Margins are used to extend or decrease the boundry of the elements edges.

Padding

Paddings are used to extend or decrease spacing inside the element's boundry.

Positioning

Now, position is quite a hectic topic. I will only cover:

  • absolute
  • relative
  • fixed
  • top/left/right/bottom

position: absolute;

This will make the current element position unbound and uneffected by siblings. With further tweaks the element will become bound the browser window (this means it can be effected by scrolling).

position: relative;

This will make the element position bound to the parent element and effected by siblings.

position: fixed

This will cause the element to be unbound by all position and become unchangable; only being positionable based of top/bottom/left/right (unable to be effected by scrolling)

top/left/right/bottom

These attributes correspond to the directions stated by them. They are relative values which depend on the type of positioning defined.

Colours

Colours aren't terribly complex, therefore I will not cover them in detail. When I introduce some sort of colouring throughout the guide it will be explained.
