Hyper Text Markup Language

What's Hyper Text Markup Language?

HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page's words and images. - investopedia

The Basic Building Blocks

HTML Tag Meaning
hn Heading Tags (n {0,1,2,3,4,5,6})
a Link Tag used for linking something else
br Single line break tag
hr Horizontal rule
div Specifies a block in the document
head Specifies the head tag which is used for front-loaded content
header Specifies the header tag which is used normally as a navigation bar
footer Specifies the foot tag which is normally used to display copyright and/or contact info
ul Unordered list
ol Ordered list
p Paragraph tag
--!...-- Comment
link Link tag is used to link external stylesheets
script Script tag is used to declare a scripting block or link an external script file
meta Meta tag is used to specify data about the page
img Image tag is used to link an image
body Body tag wraps around the entire page
form Form tag specifies a form block
input Input tag is used to get user input, normally used within forms
button Button tag is used to specify a button element

With these basic HTML tags you can build pretty much anything. As HTML is the skeletal structure of basic websites there will no guide for this topic... for now ;)
